# Inter font files for WebUI

Place the following files in this folder:

- `Inter-VariableFont_opsz,wght.ttf`
- `Inter-Italic-VariableFont_opsz,wght.ttf`

These two variable font files are enough for normal + italic rendering across font weights.

If your downloaded file names differ (for example lowercase/uppercase differences), rename them to exactly the names above.
